# Social Media -- Usage Guide

How to use #FirstMover on social media platforms.

---

## Profile Badge

Use the **Full Seal** (1080x1080) as a profile picture overlay or secondary image.

Use the **Minimal Flag** (512x512) as a small profile accent.

## Post Hashtag

Always use **#FirstMover** as a hashtag in posts about your AI innovation.

Example post:
```
We built [product] to solve [gap] in [industry].
No one else was doing it. Now we are.

#FirstMover #AI #[YourIndustry]
```

## Platform-Specific Sizing

| Platform | Recommended Asset | Size |
|----------|-------------------|------|
| LinkedIn post | Full Seal | 1080x1080 |
| LinkedIn banner | Lockup | 1200x400 |
| Twitter/X profile | Minimal Flag | 400x400 |
| Twitter/X post | Full Seal | 1080x1080 |
| Instagram post | Full Seal | 1080x1080 |
| Facebook post | Full Seal | 1080x1080 |
| YouTube thumbnail overlay | Minimal Flag | 200x200 |

## Certified vs Open on Social

- **Open badge holders:** Use the open badge assets and include attribution to Just In Time AI in your post or bio
- **Certified badge holders:** Use the certified badge assets (gold ring). No attribution required -- the certification speaks for itself
