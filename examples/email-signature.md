# Email Signature -- Embed Code

Add the #FirstMover badge to your email signature.

---

## HTML Email Signature (Open Badge -- Lockup)

```html
<table cellpadding="0" cellspacing="0" border="0">
  <tr>
    <td style="padding-top: 10px; border-top: 1px solid #e0e0e0;">
      <a href="https://github.com/dstolts/FirstMover" target="_blank" rel="noopener" style="text-decoration: none;">
        <img
          src="https://raw.githubusercontent.com/dstolts/FirstMover/main/assets/open/firstmover-open-lockup-1200x400.png"
          alt="#FirstMover"
          width="200"
          style="display: block;"
        />
      </a>
    </td>
  </tr>
</table>
```

## Tips

- Use the lockup (horizontal) for signatures -- it fits better inline
- Keep width between 150-250px for email signatures
- Some email clients strip images -- the alt text "#FirstMover" still communicates the badge
- Test in your email client before deploying (Outlook, Gmail, Apple Mail all render differently)
