## Entry Format

To submit your entry, add a row to the table above with:

| Field | Description |
|-------|-------------|
| # | Next sequential number |
| Name | Your company or product name |
| URL | Your website |
| Scope | Global, Industry, or Regional (with location) |
| Industry | Your industry vertical |
| Gap Filled | What problem you solved, why it qualifies as first mover. If competitors exist, name them and explain why you were first |
| Tier | Open or Certified |
